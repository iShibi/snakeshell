# SnakeShell

> The classic [Snake game](<https://en.wikipedia.org/wiki/Snake_(video_game_genre)>) but in a shell environment.

![SnakeShell Demo](./demo/SnakeShell_Demo.gif)

### Game Controls

| **Key** | **Action** |
| :-----: | :--------: |
|   `W`   |  Move Up   |
|   `A`   | Move Left  |
|   `S`   | Move Down  |
|   `D`   | Move Right |
| `Enter` |  Confirm   |
|  `Esc`  |    Exit    |
